# Contributing to EVRoute AI

Thank you for your interest in contributing! Here's how to get started.

## Development Setup

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/evroute-ai.git`
3. Follow the setup steps in [README.md](README.md)

## Reporting Issues

Please include:
- Steps to reproduce
- Expected vs actual behavior
- Python / Node.js version
- OS

## Pull Requests

1. Create a branch: `git checkout -b feature/your-feature-name`
2. Make your changes
3. Test that the backend starts and `/docs` loads correctly
4. Commit with a clear message: `git commit -m "feat: add feature X"`
5. Push and open a PR against `main`

## Commit Message Format

```
feat: add new feature
fix: correct a bug
docs: update documentation
refactor: restructure code without changing behavior
```
